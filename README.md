# ClawChat

Mobile-first chat webapp for async communication with an AI agent.

## Quick Start

```bash
docker compose up
```

- Client: http://localhost:3102
- Agent API: http://127.0.0.1:3100 (localhost only)
- Public API: http://127.0.0.1:3101
